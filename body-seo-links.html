Members of the Irish Moutain Running Association

Feel Healthy

Since it improves aerobic fitness, running is a great way to help improve your cardiovascular health. Plus, it burns calories and can build strength.

Outdoor Fitness

Outdoor running is wonderful physical exercise, a great way to get back to nature and release stress with other like minded people.

Social running

When we run with others, we can run farther and faster more easily. Social running can pull you out of a running funk or low mood, and inspire you to try new running events.

Beginner running

Our club is suitable for beginner runners! Having the support of a running community behind you can help you keep going.
